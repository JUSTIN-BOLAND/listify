1. Select your Spreadsheet (or create one)

Select Spreadsheet in Google Docs (or paste a url directly)

For Listify to work the spreadsheet must:

  • Include a few key columns: title, description, url, image url, type, tags – see this template spreadsheet as an example (which you can also just copy!)
  • Have a single header row at the very top of the sheet
  • Be "published". To "publish" a spreadsheet go to File menu -> Publish to the Web
